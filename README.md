# WEB早押しボタンシステム (minhaya)

## 概要

このプロジェクトは、リアルタイムなWEB早押しクイズシステムです。WebSocketを使用してリアルタイム通信を実現し、複数の参加者が同時にクイズに参加できます。

### 主な機能

- **リアルタイム早押し**: WebSocketによる低遅延な早押し競争
- **管理者機能**: クイズの開始・終了・問題管理
- **スコア管理**: 正解・誤答・一番手ボーナスによる得点計算
- **アイコン機能**: 参加者のプロフィール画像アップロード
- **レスポンシブデザイン**: PC・スマートフォンに対応

## システム構成

### サーバーサイド (Deno + Hono)
- **main.ts**: HTTPサーバーとWebSocketサーバーのメイン処理
- **types.ts**: 型定義とスコア設定
- **socket.ts**: WebSocket通信の抽象化

### クライアントサイド (Preact)
- **src/app.tsx**: メインアプリケーション
- **src/route/**: 各画面のコンポーネント
- **src/admin.tsx**: 管理者画面

### スコアリングシステム
- **正解**: +10点
- **一番手正解**: +30点（正解ボーナス+一番手ボーナス）
- **誤答**: -10点

## 起動方法

### 1. 開発モード（ウォッチモード）

```bash
# クライアントサイドのビルド（ファイル変更時に自動再ビルド）
deno task build-watch
```

別のターミナルで：

```bash
# サーバーの起動
deno task run
```

### 2. 本番モード

```bash
# クライアントサイドのビルド + サーバー起動
deno task run
```

### 3. ビルドのみ

```bash
# クライアントサイドのビルドのみ実行
deno task build
```

## アクセス方法

サーバー起動後、以下のURLにアクセスしてください：

- **参加者用**: `http://localhost:8000/`
- **管理者用**: `http://localhost:8000/admin/`（localhostからのみアクセス可能）

## 使用方法

### 参加者
1. ブラウザで `http://localhost:8000/` にアクセス
2. 名前とアイコンを設定
3. クイズ開始まで待機
4. 問題が表示されたら早押しボタンをクリック
5. 答えを入力して送信

### 管理者
1. ブラウザで `http://localhost:8000/admin/` にアクセス
2. クイズを開始
3. 問題CSVを入力
4. 問題開始ボタンをクリック
5. 参加者の回答を確認し、正解・不正解を判定
6. 問題終了後、次の問題へ進む
7. クイズを終了

## 開発者向け情報

### デバッグ
- サーバーログはコンソールに出力されます
- クライアントサイドのデバッグはブラウザの開発者ツールを使用してください

### カスタマイズ
- スコア設定: `types.ts` の `POINT_*` 定数を変更
- UI変更: `src/` 以下のコンポーネントを編集
- スタイル: `page/style.css` を編集

### 注意事項
- 管理者画面は `localhost` からのアクセスのみ許可されています
- アイコンファイルは `page/icons/` に保存され、サーバー再起動時にクリアされます
- WebSocketセッションはブラウザを閉じると自動的に切断されます
